# 桑多涅扩展备用资产生成说明

这组说明用于未来重生成或扩充资产。所有图片保持无文字、无 Logo、无水印，不直接复刻官方版式或专有徽记。角色身份以 `references/visual/sandrone/官方资材/` 为准，包装语法以 `官方视觉包装/` 为准。

## 共用视觉约束

- 档案夜色、墨蓝面板、酒红章节、旧金结构、冰蓝状态光、茶瓷暖白。
- 机械档案馆、蓝图索引、克制茶会、冰晶星芒与少量蕾丝并存。
- 避免霓虹赛博、铜管堆砌、现代屏幕、血迹恐怖、通用 AI 网络脑。
- 角色主视觉必须保留灰棕短发、蓝眼、白色机械头饰和黑白深红衣装，不新增动物耳、眼镜、现代设备或陌生武器。

## 主视觉

- `archive-overseer-wide-v1.webp`：2:1 横幅。桑多涅位于右侧档案工作台，左侧保持暗色低细节留白；背景含书架、精密机械、茶具与冰蓝状态光。
- `tea-intermission-wide-v1.webp`：16:9 横幅。安静、有秩序的茶歇，不是甜美下午茶；角色偏右，左侧可放标题，酒红织物与暖白瓷器只作局部焦点。
- `blueprint-review-wide-v1.webp`：16:9 横幅。角色在审阅机械蓝图或索引资料，对侧留白；以墨蓝图纸、旧金细线、冰蓝节点表现研究进行中。

## 透明小零件

先在均匀纯绿 `#00ff00` 背景生成，再按 imagegen Skill 的色键脚本清除背景并校验 alpha 和绿边。

- `gear-cluster-v1.webp`：三到五枚不同尺度的旧金精密齿轮，整体轮廓清楚。
- `frost-star-core-v1.webp`：冰蓝晶核、金色细框与少量星芒，无强烈霓虹。
- `tea-cup-mechanism-v1.webp`：暖白茶杯与克制机械底座，侧面完整。
- `archive-card-stack-v1.webp`：错位档案卡、金属夹与小型索引节点，不含可读文字。
- `burgundy-ribbon-loop-v1.webp`：酒红织带形成一个有方向感的环，不做礼物蝴蝶结。
- `lace-separator-segment-v1.webp`：横向短蕾丝段，两端可平滑重复。
- `index-node-v1.webp`：蓝图索引节点、菱形核心和短连接线。
- `corner-filigree-v1.webp`：单角旧金卷草与机械结构，内部留空。

## 透明动画层

同样使用纯绿底生成并清除背景。画面中心优先留空，关键图形不要被边缘硬裁。

- `gearfield-far-v1.webp`：稀疏、低对比的大型齿轮远景，适合极慢旋转或缩放。
- `frost-particles-mid-v1.webp`：金色与冰蓝霜晶、星点的中景粒子场，密度不均匀。
- `gold-node-network-v1.webp`：精细的旧金索引节点和连接线，避免电子电路观感。
- `burgundy-ribbon-near-v1.webp`：只从边缘进入的两至三道酒红织带弧线，中心净空。
- `lace-scroll-strip-v1.webp`：可横向循环的暖白蕾丝卷轴条，首尾纹理连续。

生成后必须登记到 `src/data/visualAssets.ts`，并制作联系表做整体密度和色彩检查。备用资产默认不自动挂入页面。
