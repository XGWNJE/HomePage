# 桑多涅混合资材包设计规格

- 日期：2026-07-10
- 状态：资材方向已确认，等待书面规格复核
- 范围：只生成网站视觉素材；暂不修改页面、文案或生产环境

## 目标

为首页、About、内容索引、错误/维护状态和文章对话框制作一套可共同使用的桑多涅视觉资材。普通形象保持当前官方可玩版本的完整识别特征，动作服务于“网站助手、观察者和秩序维护者”的身份；Q 版仅改变身体比例，不改变角色身份。

## 参考素材及职责

1. 官方「A Lady's Everyday Notes」横图：主视觉风格、深红/炭黑/旧金/冰蓝配色、茶会与机械秩序感。
   - 来源：https://www.hoyolab.com/article/45623659
2. 官方角色档案竖图：完整服装结构、头饰、发型、配饰、武器与身体比例。
   - 来源：https://www.hoyolab.com/article/45197167
3. 官方角色档案横图：面部、瞳色和上半身细节的补充参照。
   - 来源：https://www.hoyolab.com/article/45197167

生图时，横图作为风格与氛围参考，竖图作为角色结构参考，角色档案横图作为面部补充参考。不得把参考图中的官方文字、Logo 或版式复制进新素材。

## 全局角色不变量

- 浅灰棕色短发和官方刘海、侧发结构保持一致。
- 蓝色瞳孔、面部轮廓和冷静克制的神态保持一致。
- 保留大型白色机械感头饰、黑色颈部结构、白黑红裙装、金色框架和冰蓝发光机械细节。
- 服装层级、袖口、手套、裙摆、红色飘带和主要配饰不得随意简化或重新设计。
- 普通形象保持官方身材比例；Q 版使用约 2.5 头身，但脸、发型、头饰和服装仍能立即辨认为桑多涅。
- 动作克制、精准、略显挑剔；不卖萌、不做夸张偶像姿势、不使用恋爱或主仆暗示。
- 不添加动物耳朵、眼镜、现代电子产品、陌生武器、额外角色或非官方服装。
- 所有素材禁止文字、Logo、签名和水印，不暗示 HoYoverse 为本站背书。

## 资材清单

### 1. 首页引导图 `hero-guide`

- 用途：首页首屏主视觉。
- 构图：16:9 横向；桑多涅位于画面右侧约 40% 区域，三分之二身或全身；左侧保留干净文案安全区。
- 动作：一手轻抬，像在示意访客进入索引；另一手自然靠近机械资料装置或茶杯。
- 场景：深色档案室与精密机械工作台的结合，齿轮和冰蓝光线只作背景层次。
- 情绪：冷静、准确、已经替访客整理好入口。
- 输出：高质量横图，项目文件建议为 `public/image/sandrone/hero-guide-v1.webp`。

### 2. About 观察者图 `about-observer`

- 用途：About 主视觉及人物关系介绍。
- 构图：4:5 竖向；人物坐在茶桌或机械工作台旁，身体朝向站长内容区域，视线轻微转向访客。
- 动作：整理纸质档案、机械图纸或索引卡；姿态自然，不摆拍。
- 场景：暖白纸张、深红织物、黑色机械和少量冰蓝状态光，体现外冷内稳。
- 情绪：观察站长工作、略有挑剔，但正在认真协助。
- 输出：高质量竖图，项目文件建议为 `public/image/sandrone/about-observer-v1.webp`。

### 3. 内容索引立绘 `index-assistant`

- 用途：文章、标签、项目列表的空白区或状态卡。
- 构图：独立全身或四分之三身，1:1 安全画布，四周留足透明边距。
- 动作：左手持资料板，右手调整悬浮索引卡或小型机械指针。
- 情绪：正在核对分类，神态专注。
- 背景：先使用纯色键控背景生成，再处理为透明 PNG。
- 输出：保留透明通道的无损 WebP，`public/image/sandrone/index-assistant-v1.webp`。

### 4. 错误/维护立绘 `maintenance-assistant`

- 用途：404、加载失败、维护和无结果状态。
- 构图：独立半身或四分之三身，1:1 安全画布。
- 动作：检修一枚错位齿轮或失序机械模块；工具和装置保持简洁，不遮挡面部与服装识别点。
- 情绪：轻微不耐烦，但动作仍可靠；不生气咆哮，不做喜剧摔倒。
- 背景：先使用纯色键控背景生成，再处理为透明 PNG。
- 输出：保留透明通道的无损 WebP，`public/image/sandrone/maintenance-assistant-v1.webp`。

### 5. 文章对话框 Q 版 `dialog-chibi`

- 用途：文章内桑多涅短评或对话框角落装饰。
- 构图：约 2.5 头身，半身偏全身，1:1 安全画布；轮廓清楚，缩至 80–160 px 仍可识别。
- 动作：怀抱小型记录板，另一手握笔或轻点记录板边缘。
- 表情：平静、略带审视，不微笑卖萌；允许非常轻微的无奈感。
- 简化规则：减少细小纹样，但保留发型、蓝眼、白色头饰、黑白红衣装、金色与冰蓝机械特征。
- 背景：先使用纯色键控背景生成，再处理为透明 PNG。
- 输出：保留透明通道的无损 WebP，`public/image/sandrone/dialog-chibi-v1.webp`。

## 视觉统一规则

- 场景图使用深红、炭黑、暖白、旧金和冰蓝；避免粉色梦幻、霓虹赛博朋克和纯黑恐怖风。
- 机械元素作为秩序与结构的暗示，不把桑多涅写成普通机器人。
- 光线由暖白环境光和低强度冰蓝机械光组成，人物肤色保持自然。
- 首页与 About 的绘制质感一致；独立立绘与 Q 版使用同一套颜色和线条边缘逻辑。
- 文案安全区不得被飘带、机械臂、强光或复杂装饰侵入。

## 生成与验收流程

1. 使用内置生图，以三张官方素材共同作为角色与风格参考。
2. 每项资材单独生成，不使用一张拼图裁切成多个资产。
3. 检查面部、头饰、服装、手部、飘带、机械结构和配色；身份漂移则重新生成。
4. 检查网页构图安全区，并分别模拟桌面和移动端裁切。
5. 三张独立立绘使用纯色键控背景生成，本地去背后验证透明角、头发边缘、白色头饰和冰蓝发光部件。
6. 将最终版本保存到 `public/image/sandrone/`，保留清晰来源记录，不覆盖现有用户素材。

## 验收标准

- 不看说明即可明确辨认为桑多涅，而非“桑多涅风格原创角色”。
- 五张图之间脸型、发型、服装、头饰、色彩和绘制质感一致。
- 普通版无明显手指、肢体、服装层级或机械结构错误。
- 主页左侧和 About 正文区域拥有足够可读空间。
- Q 版缩小后仍能辨认，透明边缘无明显键控色残留。
- 无文字、Logo、水印、额外角色或官方背书暗示。
