# HomePage 项目规则

本项目是 `xgwnje.cn` 新方向的独立起点，不继承旧 `D:\ObjectCode\My-server\Home-page` 的实现。

## 设计方向

- 目标是“带有抽象信号气质的个人内容索引站”，不是字面电台、播客或控制台界面。
- 首页结构优先服务内容阅读和浏览；抽象信号、频谱、扫描线只作为氛围和动效语言。
- 不使用明显收音机、麦克风、耳机、天线、唱片、磁带等直白图形。
- 内容类型先保持：笔记、影像、作品。

## Development

When starting the dev server, use background mode:

```powershell
npm run dev -- --host 127.0.0.1
```
